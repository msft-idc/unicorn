# 🦄 Unicorn

Project Unicorn - A magical utility that brings some sparkle to your day!

## Description

Unicorn is a simple Python utility that generates magical unicorn art and inspirational messages. Because sometimes you just need a little magic in your terminal.

## Features

- Generate ASCII unicorn art
- Display inspirational unicorn-themed messages
- Colorful terminal output
- Simple command-line interface

## Usage

```bash
python unicorn.py
```

## Installation

No special installation required! Just run the script with Python 3.

```bash
git clone https://github.com/msft-idc/unicorn.git
cd unicorn
python unicorn.py
```

## Requirements

- Python 3.6+
- No external dependencies required
