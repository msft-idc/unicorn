| Step | Action | Description |
|------|--------|-------------|

| 2️⃣ | **🌟 Create** | Create a feature branch: `git checkout -b feature/amazing-feature` |

| 4️⃣ | **🚀 Push** | Push to the branch: `git push origin feature/amazing-feature` |
| 5️⃣ | **🎉 Open** | Open a Pull Request |
